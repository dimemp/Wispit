# Wispit Ventures

Source for **[Wispit Ventures](https://wispitventures.com/)** — the personal studio of Dimitris Dimosiaris. The site presents a venture studio focused on **technical leadership for startups and SMEs**: mentorship, MVP development, and fractional CTO work to ship web products. It includes sections for services, products, and background on the studio.

Built as a static site with HTML, Bootstrap, and custom CSS and JavaScript.
